# Next Word Prediction using LSTM

A deep learning based **Next Word Prediction** model built using **TensorFlow/Keras and LSTM networks**.  
The model learns contextual relationships between words from a dataset of quotes and predicts the most probable next word given an input sequence.
---

## Features

- Built an **LSTM-based language model** for next-word prediction
- Converted raw text into **n-gram sequences** for supervised learning
- Used **word embeddings** to learn semantic representations
- Implemented **Bidirectional LSTM architecture** for better context understanding
- Added **Dropout regularization** to reduce overfitting
- Evaluated performance using **Accuracy, Top-5 Accuracy, and Perplexity**
- Implemented a **text generation pipeline** for sentence completion

## Evaluation Metrics

The model performance is evaluated using:

- **Accuracy**
- **Top-5 Accuracy**
- **Perplexity**

These metrics help evaluate both **prediction correctness** and **language modeling quality**.
